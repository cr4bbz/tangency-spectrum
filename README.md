# Tangency Spectrum

Research notes on point distributions on the interval \([0,1]\), starting from
equal spacing and moving toward projective tangency dynamics, Stern-Brocot
coding, zeta functions, and multifractal boundary measures.

The motivating contrast is:

\[
\text{additive equal spacing}
\qquad\text{versus}\qquad
\text{projective equal spacing}.
\]

In the dyadic model, refinement halves every interval. In the tangency model,
the projected skeleton chain is evenly spaced in the logit coordinate

\[
\eta(p)=\log \frac{p}{1-p}.
\]
